# 安装

目前**TorchAir暂未提供独立软件包**，而是作为Ascend Extension for PyTorch的三方库，随着torch\_npu包一起发布。请直接安装torch\_npu插件，即可使用TorchAir。

torch\_npu的安装操作具体参考《Ascend Extension for PyTorch 软件安装指南》，请保证与CANN相关包的版本匹配（参见《版本说明》），否则功能可能无法正常使用。

**需要注意的是**，当安装的torch\_npu版本为7.3.0及之后版本，均可正常使用TorchAir，对于其他torch\_npu版本请参见对应版本文档中的安装介绍。
