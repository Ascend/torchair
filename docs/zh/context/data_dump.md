# data\_dump

## 功能说明

使用算子输入输出dump功能（图模式）时，可以配合本接口，在脚本中指定dump范围内的算子。

## 函数原型

```python
data_dump()
```

## 参数说明

无

## 返回值说明

无

## 约束说明

-   本功能支持max-autotune模式。
-   max-autotune模式下，需要配合[算子输入输出dump功能（图模式） ](算子输入输出dump功能（图模式）.md)使用。
