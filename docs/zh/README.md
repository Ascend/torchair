# 项目文档

## 简介

此目录提供[Ascend Extension for PyTorch文档中心](https://hiascend.com/document/redirect/pytorchuserguide)中《PyTorch图模式使用(TorchAir)
》文档的源文件信息，通过[目录文件](_menu_torchair.md)可快速查阅该文档的全部信息。

## 目录说明

关键目录结构如下：

```txt
├── npugraph_ex                # 基于npugraph_ex后端的图模式功能介绍
├── ascend_ir                  # 基于GE的图模式功能介绍
├── custom_op_graph            # 自定义PyTorch算子接入TorchAir
└── appendix                   # 常见案例等附录信息
```
